Реализация дипломной работы на тему:\
"Проектирование и разработка корпоративной информационной системы управления обращениями пользователей (Help Desk)"

Как запустить приложение:

1. Сделать скрипт запуска исполняемым:\
   1.1. На linux: chmod +x ./docker/postgres/init/01-init-roles.sh\
   1.2. На Windows:\
   1.2.1. Проверить окончания строк скрипта, должно быть "LF" - иначе приложение не запустится.\
   Если окончания "CRLF", сменить через VSCode (открыть файл - правый нижний угол - CRLF сменить на LF)\
или другим удобным способом.\
   1.2.2. Проверить, что файл существует, этого достаточно для запуска: dir .\docker\postgres\init\01-init-roles.sh
2. Первичный запуск (или если нужно пересобрать приложение): docker compose --build
3. Запуск остановленного приложения (без персборорки): docker compose up -d
